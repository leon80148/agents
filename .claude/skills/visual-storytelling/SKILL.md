# 視覺敘事 (Visual Storytelling)

此 Skill 提供視覺敘事、資訊圖表設計、簡報製作和數據視覺化的專業知識。當需要將複雜想法轉化為引人入勝的視覺故事時，請運用這些原則和技巧。

## 視覺敘事原則
1. **清晰優先**：如果不清晰，就不聰明
2. **情感連結**：事實告知，故事銷售
3. **漸進揭示**：逐步展現複雜性
4. **視覺一致性**：統一風格建立信任
5. **文化意識**：符號在不同地方有不同意義
6. **無障礙**：每個人都值得理解

## 故事結構框架
```
1. 鉤子（抓住注意力）
   - 驚人統計
   - 相關問題
   - 引人入勝的問題

2. 背景（設定舞台）
   - 當前情況
   - 為什麼重要
   - 涉及的利害關係

3. 旅程（展示轉變）
   - 面臨的挑戰
   - 發現的解決方案
   - 取得的進展

4. 解決（交付回報）
   - 達成的結果
   - 實現的好處
   - 未來願景

5. 行動呼籲（驅動行為）
   - 明確的下一步
   - 引人注目的理由
   - 簡單的前進路徑
```

## 數據視覺化工具包
- **比較**：長條圖、柱狀圖
- **組成**：圓餅圖、堆疊長條圖、樹狀圖
- **分佈**：直方圖、箱形圖、散佈圖
- **關係**：散佈圖、氣泡圖、網路圖
- **時間變化**：折線圖、面積圖、甘特圖
- **地理**：分層著色圖、符號地圖、流量圖

## 資訊圖表佈局模式
```
時間軸佈局：
[開始] → [事件 1] → [事件 2] → [結束]

比較佈局：
| 選項 A | vs | 選項 B |
|  優點  |    |  優點  |
|  缺點  |    |  缺點  |

流程流程：
輸入 → [處理] → 輸出
  ↓        ↓         ↓
細節     細節      細節

統計故事：
大數字
支持統計 1 | 統計 2 | 統計 3
背景和解讀
```

## 敘事色彩心理學
- **紅色**：緊迫、熱情、警告
- **藍色**：信任、穩定、平靜
- **綠色**：成長、健康、金錢
- **黃色**：樂觀、注意、謹慎
- **紫色**：奢華、創意、神秘
- **橙色**：能量、熱情、親民
- **黑色**：精緻、力量、優雅
- **白色**：簡約、乾淨、空間

## 視覺故事中的字體排版
```
Display: 48-72px - 大影響力聲明
Headline: 32-40px - 區段標題
Subhead: 24-28px - 支持要點
Body: 16-18px - 詳細資訊
Caption: 12-14px - 額外背景
```

## 圖示設計原則
- 一致的線條寬度（通常 2-3px）
- 簡化形式（移除不必要細節）
- 清晰隱喻（即時可識別）
- 統一風格（線框、填充或雙色調）
- 可縮放設計（所有尺寸都有效）
- 文化中立（避免特定參考）

## 插圖風格指南
```
角色設計：
- 比例：1:6 頭身比
- 特徵：簡化但有表現力
- 多樣性：包容性代表
- 姿勢：動態且符合情境

場景構圖：
- 前景：主要動作/角色
- 中景：支持元素
- 背景：情境/環境
- 深度：使用重疊和縮放
```

## 故事動畫原則
1. **進入**：元素有目的地出現
2. **強調**：關鍵點脈動或縮放
3. **過渡**：平滑的狀態變化
4. **退出**：清晰的完成信號
5. **時機**：大多數動畫 200-400ms
6. **緩動**：自然的加速/減速

## 簡報投影片範本
```
標題投影片：
[大膽聲明]
[支持副標題]
[細微視覺元素]

數據投影片：
[陳述洞察的清晰標題]
[佔 60% 空間的視覺化]
[突出顯示的關鍵要點]

比較投影片：
[問題或選擇]
選項 A | 選項 B
[視覺呈現]
[結論]

故事投影片：
[場景插圖]
[敘事文字覆蓋]
[情感連結]
```

## 社群媒體優化
- **Instagram**：1:1 或 4:5 比例，大膽色彩
- **Twitter**：16:9 比例，小尺寸可讀
- **LinkedIn**：專業語調，數據導向
- **TikTok**：9:16 比例，適合動態
- **Pinterest**：2:3 比例，勵志風格

## 無障礙檢查清單
- [ ] 色彩對比符合 WCAG 標準
- [ ] 文字縮放時保持可讀
- [ ] 動畫可暫停/停止
- [ ] 替代文字描述視覺內容
- [ ] 顏色不是唯一的資訊載體
- [ ] 互動元素可用鍵盤操作

## 視覺故事測試
1. **5 秒測試**：主要訊息清楚嗎？
2. **瞇眼測試**：層次有效嗎？
3. **灰階測試**：沒有顏色能運作嗎？
4. **行動測試**：小螢幕可讀嗎？
5. **文化測試**：跨情境適當嗎？
6. **無障礙測試**：每個人都能使用嗎？

## 常見視覺故事錯誤
- 資訊過載（一次太多）
- 裝飾優於傳達
- 視覺語言不一致
- 對比度和可讀性差
- 缺少情感連結
- 流程或順序不清
- 文化不敏感

## 交付物格式
- **靜態**：PNG、JPG、PDF
- **向量**：SVG 用於縮放
- **互動**：HTML5、Lottie 動畫
- **簡報**：Keynote、PowerPoint、Google Slides
- **社群**：為各平台調整尺寸
- **印刷**：高解析度帶出血

## 快速創作工具
- **Figma**：協作設計
- **Canva**：快速範本
- **D3.js**：數據視覺化
- **After Effects**：動態圖形
- **Lottie**：輕量動畫
- **Flourish**：互動圖表
